About

Project Information

MCA Mini Project — Care Transition Efficiency & Placement Outcome Analytics

🎯 Objective

To analyze and visualize hospital care transition patterns, predict readmission risk, and optimize patient placement outcomes using data analytics techniques. The project helps healthcare administrators identify bottlenecks and high-risk patients before discharge.

⚙️ Technologies Used

Python 3.10PandasNumPy MatplotlibSeabornScikit-learn StreamlitGoogle ColabHTML/CSS/JS SQL (SQLite)GitHub Pages

📦 Dataset Details

  • 2,847 synthetic patient records
  • 12 features per record (demographics, clinical, outcome)
  • 6 departments, 4 discharge destinations
  • Generated using Python Faker + NumPy
  • Date range: Jan 2024 – Dec 2024
  • No real patient PII included

🔄 Workflow

1
Data Generation — Synthetic dataset with realistic distributions
2
EDA — Exploratory analysis in Google Colab
3
ML Modeling — Logistic Regression for readmission risk
4
Dashboard — Streamlit + HTML interactive dashboards
5
Deployment — Streamlit Cloud + GitHub Pages
